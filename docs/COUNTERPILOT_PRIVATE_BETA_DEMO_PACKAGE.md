# Counterpilot Private Beta Demo Package

This package is the first merchant-conversation bundle for Counterpilot Private
Beta v0. It is meant to support a short, honest product conversation, not a
launch campaign.

## Demo Goal

Show that Counterpilot can run a manual Shopify make-an-offer flow and report
observed mature margin after payment, refunds, returns, and maturity.

## Current Product Claim

Counterpilot is a Shopify make-an-offer app that tracks negotiated orders
through payment, refunds, returns, and maturity so merchants can see true mature
margin, not just accepted offers.

## What The Demo Should Prove

- A shopper can submit an offer from a product page.
- A merchant can manually accept, counter, or decline.
- Checkout is created only after buyer acceptance.
- Paid, refund, return, and maturity events are tracked.
- The merchant report shows observed mature margin with costs and refunds.
- Demo and test evidence is marked `production_evidence: false`.

## What The Demo Must Not Claim

- No AI negotiation.
- No automated pricing.
- No conversion-lift claim.
- No profit-lift claim.
- No recovered-revenue claim.
- No counterfactual claim about what would have happened without Counterpilot.
- No recommendation-model claim.

## Merchant Conversation Target

Use this package to start three private-pilot conversations with Shopify
merchants who already care about negotiated sales, margin, refunds, and returns.

The question is not whether they want AI. The question is whether a manual-first
offer flow with mature-margin reporting is worth trying in their store.
